Spin up the project med Docker:
1. docker build -t vloris_webapp_frontend .

2. docker run -d -p 3002:3002 --name run_vloris_webapp_frontend vloris_webapp_frontend

3. docker logs --tail 1000 -f run_vloris_webapp_frontend

You can also do all 3 of them in one: 

docker build -t vloris_webapp_frontend . && docker run -d -p 3002:3002 --name run_vloris_webapp_frontend vloris_webapp_frontend && docker logs --tail 1000 -f run_vloris_webapp_frontend


TO STOP IT:

1.docker stop run_vloris_webapp_frontend

2.docker rm run_vloris_webapp_frontend

You can also do both of them in one:
docker stop run_vloris_webapp_frontend && docker rm run_vloris_webapp_frontend
